# Anveet Pal - Personal Portfolio Website

A responsive, professional personal portfolio website designed to showcase my skills in Computer Science, Python, Machine Learning, and Data Analytics. This project features a modern dark-mode aesthetic with smooth animations and interactive elements.

## Built With

* **HTML5** - Semantic markup for structure.
* **CSS3** - Custom styling, CSS Variables, Flexbox/Grid, and Animations.
* **JavaScript (ES6+)** - DOM manipulation, Modal logic, and dynamic UI effects.
* **Font Awesome** - For social media and UI icons.
* **Google Fonts** - Using 'Inter' for clean typography.
